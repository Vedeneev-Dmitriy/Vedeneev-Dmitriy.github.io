Outpost Firewall

Сайт производителя

Рис 1. Главное меню

 Outpost Firewall - файерволл, предназначенный для нетребовательных пользователей, обладает базовым функционалом. Программа не требует настройки и сразу после установки готова к работе.

Бесплатная версия ограничена по времени - 30 дней. После этого требуется оплатить лицензию - 900 руб в год.

Рис 2. Вкладка "Брандмауэр"

 Во вкладке "Брандмауэр" отображаются процессы, потребляющие трафик. Внизу имеются кнопки для прерывания подключений и настройки правил для каждого процесса.

Рис 3. Редактор правил

 Редактор правил представляет собой набор простых инструментов по управлению доступом приложений к сети. Вкладка "Общие" позволяет пользователю выбрать один из трех режимов работы брандмауэра с данным приложением. Остальные вкладки позволяют настроить сетевые правила, доступ приложения к конкретным действиям в системе.

Рис 4. Вкладка "Инструменты"

 В пункте "Инструменты" находятся различные статистические данные.

Рис 5. Вкладка "Настройки"

 В пункте "Настройки" представлены основные модули программа. По нажатию на каждый из модулей он либо включается, либо выключается.

Рис 6. Ползунки. Настройки Anti-leak

 В подпункте "Дополнительные настройки" пользователь с помощью ползунков может настроить уровень безопасности, обеспечиваемый каждым из модулей.

Рис 7. Меню настроек "Реклама и сайты"

 Также в программе присутствует модуль, обеспечивающий блокировку вредоносных сайтов и рекламы. Есть возможность создать список сайтов, доступ на которые будет запрещен.

Рис 8. Всплывающее окно программы

 Основное взаимодействие пользователя с программой осуществляется посредством всплывающего меню, позволяющего применить конкретную настройку к приложению, выходящего в сеть. Также есть возможность сохранить принятое решение, чтобы при дальнейшей работе данный процесс не требовал повторного подтверждения от пользователя.
