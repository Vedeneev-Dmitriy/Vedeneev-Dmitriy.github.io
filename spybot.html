SpyBot

Сайт производителя

Рис 1. Главное меню

 SpyBot — программа, предназначенная для предотвращения и устранения заражения компьютера пользователя шпионским программным обеспечением (spyware), троянами, malware, scumware, вредоносными дополнениями к браузеру, компонентами, отслеживающими предпочтения пользователя без его согласия. Приложение анализирует системные процессы и выявляет вредоносное ПО (программы-шпионы, так называемые руткиты), инструментами которого могут быть похищены конфиденциальные данные.

Рассматриваемая редакция приложения распространяется бесплатно. Однако на сайте производителя можно приобрести улучшенные версии данного ПО - SpyBot Home (добавлен антивирус) по цене 16 $ и SpyBot Pro (добавлены антивирус и средство создания загрузочных дисков) по цене 28 $.

В программе представлены три набора инструментов, это базовые, расширенные и профессиональные инструменты (доступны только в платных версиях). [6]

Рис 2. Пункт "Создатель отчета"

 Создатель отчета. Открыв это окно, пользователь сможет создать Log-файл с подробным описанием системы и потом, если понадобится, отправить в службу поддержки SpyBot.

Рис 3. Пункт "Настройки"

 Настройки. Здесь можно изменить настройки, от языка самой программы до включения/выключения отдельных модулей программы.

Рис 4. Пункт "Инструменты стартовой загрузки"

 Инструменты стартовой загрузки. Представленные в данной вкладке интсрументы позволяют пользователю контролировать процессы, запускаемые при старте системы. Также здесь представлен менеджер процессов, по своим функциям дублирующий стандартный диспетчер задач Windows.

Рис 5. Пункт "Rootkit сканер"

 Rootkit - сканер. Один из важнейших интсрументов SpyBot. При помощи данного модуля осуществляется поиск руткитов (rootkits) – утилит, используемых хакерами для сокрытия своей вредоносной деятельности на зараженном компьютере. Для этого они модифицируют операционную систему компьютера. Руткиты не только скрывают свое собственное присутствие, но также и действия, которые предпринимает злоумышленник. Руткит может скрывать присутствие других вредоносных программ на компьютере, изменяя файловые данные, ключи реестра или активные процессы. ***********

Рис 6. Пункт "Иммунизация"

 Иммунизация. Позволяет провести иммунизацию некоторых установленных браузеров и файла Hosts. Для браузеров это означает, что исправляются уязвимости плагинов, файлов куки и других элементов браузера. А иммунизация файла Hosts означает, что в этот файл будут записаны адреса тысяч сайтов, находящихся в черном списке программы и дальейшем в эти сайты доступ будет закрыть.

Рис 7. Пункты "Сканирование системы", "Сканирование файлов", "Подготовка белого списка"

 Сканирование системы, файлов, создание "белого списка". В данной редакции программы нет защиты в режиме реального времени, она представлена только в платных версиях программы. Поэтому сканирование ПК происходит в ручном режиме. Сканер SpyBot считается одним из самых качественных, в сети много отзывов, в которых говорится, что пользователи обезвредили с помощью этой программы много угроз, которые не смогли найти многие другие антишпионы.

Отличие между сканером системы и сканером файлов состоит в том, что сканер файлов проверяет только те файлы, которые выбрал сам пользователь. А сканер системы проверяет весь ПК полностью.

С помощью белого списка пользователь может указать файлы, которые программа будет игнорировать при сканировании.
