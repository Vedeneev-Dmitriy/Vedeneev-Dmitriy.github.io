Выбор операционноый системы

Операционная система — комплекс взаимосвязанных программ, предназначенных для управления ресурсами компьютера и организации взаимодействия с пользователем. В логической структуре типичной вычислительной системы операционная система занимает положение между устройствами с их микроархитектурой, машинным языком и, возможно, собственными (встроенными) микропрограммами (драйверами) — с одной стороны — и прикладными программами с другой.

Windows

Windows — семейство операционных систем (OC) корпорации Microsoft, ориентированных на применение графического интерфейса при управлении.[4]

Плюсы:

  • • Хороший стартовый комплект ПО.

  • • Высокая совместимость.

  • • Простая и понятная установка.

  • • Безопасная персонализация.

  • • Коллекция драйверов.

  • • Широкое сообщество пользователей.

  • • Простое и готовое решение, не требующее жесткой привязки к железу или множества настроек перед началом работы.

Минусы:

  • • Плохая обучающая составляющая.

  • • Наиболее развита в области вирусного ПО.

  • • Неочевидные названия или описания для некоторых функций.

  • • Синий экран смерти. (Обычному пользователю зачастую трудно как-либо среагировать на подобную внезапную ошибку, и в случае неподготовленности все несохраненные данные просто теряются. При наличии подобных форс-мажоров трудно говорить о высокой стабильности).

Mac OS

Mac OS — семейство проприетарных операционных систем производства корпорации Apple. Разработана для линейки персональных компьютеров Macintosh.[4]

Плюсы:

  • • Графическая оболочка, с одной стороны, красива, с другой стороны, не «перегружена» спецэффектами.

  • • Удобный для пользователя интерфейс с множеством оригинальных функций, облегчающих работу с компьютером.

  • • Высокая безопасность в связи с меньшим количеством вирусов из-за неширокой распространенности этой операционной системы.

  • • Совместимость основных форматов (DOC, RTF, PDF, JPEG, MP3, HTML и др.) с другими компьютерами.

  • • Стабильная работа (Mac OS разработана специально для компьютеров Macintosh; интегрирование в операционную систему большинства драйверами для стороннего оборудования; высокая совместимость компонентов, минимальные трудности с охлаждением).

  • • Простота в установке и удалении приложений.

  • • Возможность установки Windows в качестве второй операционной системы (с помощью утилиты Boot Camp на всех компьютерах Macintosh с процессором Intel).

Минусы:

  • • Меньший по сравнению с Windows выбор программ.

  • • Меньшее количество игр

  • • Меньшая гибкость в управлении размерами и расположением панелей.

  • • Более высокая стоимость. Компьютеры Apple стоят дороже примерно в 2 раза, чем РС аналогичной конфигурации.

Linux

Linux— семейство Unix-подобных операционных систем на базе ядра Linux, включающих тот или иной набор утилит и программ проекта GNU, и, возможно, другие компоненты. Как и ядро Linux, системы на его основе как правило создаются и распространяются в соответствии с моделью разработки свободного и открытого программного обеспечения. Linux-системы распространяются в основном бесплатно в виде различных дистрибутивов — в форме, готовой для установки и удобной для сопровождения и обновлений, — и имеющих свой набор системных и прикладных компонентов, как свободных, так возможно и собственнических. [4]

Плюсы:

  • • ПО бесплатное - нет рекламы.

  • • Возможна более тонкая настройка системы.

  • • Сразу после установки предустановлены приложений в основном для разработки и вероятно консольные.

  • • Не нужно ходить по сайтам и рыскать ПО, ведь для этого есть репозитории.

  • • Есть поддержка приложений win.

  • • Драйвера идут сразу "из коробки".

  • • Простота обновления

  • • Возможна более точная настройка (кастомизация) встроенными средствами ОС.

  • • Антивирус не нужен.

Минусы:

  • • Cистема рассчитана не для обычного пользователя.

  • • Придется начать работать в консоли.

  • • Отсутствие возможности использовать некоторые специализированные профессиональные приложения в этой ОС.

  • • Отсутствие поддержки некоторого оборудования.

ПО, рассматриваемое в данной курсовой работе, предназначено для ОС Windows, т.к. на данный момент - это самая распространенная операционная система в мире, таким образом у пользователей не будет проблем с установкой данного ПО. Однако, в данной работе также рассмотрен файловый менеджер ОС Mac OS. Это сделано для того, чтобы познакомить пользователей с небольшой часть данной ОС, которая устанавливается только на компьютеры фирмы Apple. А позволить их себе могут далеко не все.
