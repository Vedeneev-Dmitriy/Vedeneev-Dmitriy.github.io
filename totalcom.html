Total Commander

Сайт производителя

Рис 1. Total Commander

 Total Commander - один из старейших представителей файловых менеджеров. К сожалению, программа не бесплатна и требует лицензии - 600 руб. В меню Total Commander можно видеть много вкладок, отвечающих за различные функции. Основное окно программы по умолчанию разделено на две части. В первой, как правило, системный диск, но пользователь может переключиться на любой другой, во второй части какой-то другой дисковый накопитель. Между этими окнами пользователь может переносить файлы.

Рис 2. Выбор дисков

 Изменить диски можно мышкой или клавиатурой, нажав Alt+F1 или Alt+F2 для левой и правой панели тотала соответственно. Или же выбрать диски на верхней панели.

В двухпанельном отображении намного удобнее копировать или перемещать файлы. В одной панели ставим откуда копировать – в другой панели куда копировать.

Рис 3. Нижняя панель

 Внизу есть панель, где подписаны основные операции и горячие клавиши для этих операций. Перемещение между двумя панелями осуществляется клавишей TAB.

Рис 4. Настройки

 Рассмотрим настройки файлового менеджера. На первой вкладке сосредоточены одни из самых важных настроек, определяющих внешний вид и функциональные возможности окна файлового менеджера. Для внесения изменений необходимо установить/снять галочку с выбранной натсройки.

Рис 5. Поиск

 Поиск по файлам в Total Commander во многом превосходит аналогичный поиск средствами операционной системы. Одна из интересных особенностей - поиск по содержимому файлов. Например, пользователь может ввести текст из какого-то документа, а программа найдет этот документ.

Рис 6. Возможности по работе с файлами

 Total Commander обладает широкими возможности по работе с файлами, помимо базовых операций копирования или вставки, есть такие, как кодирование и декодирование файлов и архивов, просмотр огромного количество системных файлов и параметров, чтение огромного количества различных форматов файлов, как тестовых, так и графических. Большой список операций с архивами, особенно с малоизвестными, которые не могут открыть простые архиваторы.

Рис 7. FTP

 FTP клиент, интегрированный в этот файловый менеджер, позволяет подключаться к серверу по протоколу ФТП и работать с его файлами точно так, как будто бы они находятся на вашем компьютере. Для выбора подключения достаточно будет нажать CTRL+F, а если еще ни одного соединения не настроено, то еастройку можно осуществить, нажав кнопку «Добавить».

Другие возможности Total Commander:

  • • Показать все файлы без подкаталогов - показ файлов одним списком, без необходимости открывать каталоги.

  • • Групповое переименование - переименование выбранных файлов по определенной маске.

  • • Внутренние ассоциации - позволяет настроить ассоциации с любыми программами для файлов.

  • • Поддержка плагинов - наборов мелких утилит для самых разных задач, которые расширяют возможности Total Commander.
